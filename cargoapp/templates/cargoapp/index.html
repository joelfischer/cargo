Welcome to cargo game management.

1. Set up the readers.

In the admin pages, create 'locations'.

2. Set up some tags.

Using the set up reader, scan tags and assign an alias to each tag.

3. Create game sessions.

Create game session in the admin pages.

4. Register players.

Record players details and assign a tag and game session to each player.

5. Make a game session live.

Choosing the session loads players and makes the session 'live'.

6. Observe players checking in.

Once the game started, go to check ins to keep an eye on what's happening.

7. Trigger phone calls.

Once you observe a certain check in, go to calls to trigger specific phone calls to players.
